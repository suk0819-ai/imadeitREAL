
# 로또 번호 추첨기

## 개요

이 프로젝트는 사용자가 버튼을 클릭하여 1부터 45까지의 숫자 중 6개의 고유한 로또 번호를 생성할 수 있는 간단한 웹 애플리케이션입니다. 이 애플리케이션은 최신 웹 표준을 사용하여 구축되었으며, 재사용 가능한 UI를 위해 웹 컴포넌트를 활용합니다.

## 디자인 및 기능

*   **UI:**
    *   깨끗하고 현대적인 디자인.
    *   생성된 각 번호를 표시하는 다채로운 공.
    *   "번호 생성" 버튼.
    *   모바일 반응형 레이아웃.
*   **기능:**
    *   사용자가 버튼을 클릭하면 6개의 고유한 랜덤 번호를 생성합니다.
    *   생성된 번호를 화면에 시각적으로 표시합니다.
    *   생성된 각 공에 약간의 애니메이션 효과를 추가합니다.

## 현재 구현 계획

1.  **`index.html`**:
    *   페이지 제목을 "로또 번호 추첨기"로 설정합니다.
    *   `style.css`와 `main.js`를 연결합니다.
    *   `<lotto-generator>` 웹 컴포넌트를 페이지에 추가합니다.
2.  **`style.css`**:
    *   전체 페이지에 대한 기본 스타일(배경색, 글꼴 등)을 설정합니다.
    *   애플리케이션의 중앙 정렬을 위한 스타일을 추가합니다.
3.  **`main.js`**:
    *   `<lotto-generator>`라는 이름의 커스텀 엘리먼트를 정의합니다.
    *   **Shadow DOM**을 사용하여 컴포넌트의 스타일과 구조를 캡슐화합니다.
    *   컴포넌트 내에 "번호 생성" 버튼과 번호를 표시할 영역을 만듭니다.
    *   버튼 클릭 시 1부터 45 사이의 고유한 랜덤 번호 6개를 생성하는 로직을 구현합니다.
    *   생성된 번호를 다채로운 공 형태로 동적으로 화면에 표시하는 함수를 구현합니다.
